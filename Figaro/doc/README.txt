Thanks for downloading the Figaro probabilistic programming language.

Figaro requires Scala 2.10 or later. You can download Scala at http://www.scala-lang.org.

To use Figaro, link in lib/figaro.jar and lib/jsci-core.jar into your Java or Scala application. This can be
achieved by adding these files to your classpath, or, if you are using an IDE, adding them to the libraries
for your project. If you are running with Java (rather than Scala), you will also need to link in scala-library.jar.

Please read the tutorial in doc/Figaro Tutorial.docx.

For more information, view the scaladoc in doc/scaladoc/index.html.

Charles River Analytics provides probabilistic modeling services using Figaro and other tools. 
For information, please visit www.cra.com.
