

Figaro 1.0.3
============

Added score method to element that computes the MH ratio between the proposed state of the element and the original state. Documented score method in tutorial.

Added computation of values of reference elements, enabling them to be used in algorithms such as variable elimination.

Changed MakeList not to inherit from Chain.
